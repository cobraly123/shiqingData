DeepSeek R1 是幻方量化旗下 DeepSeek 于 2025 年 1 月 20 日发布的专注复杂推理的大模型系列，以纯强化学习 + 多阶段训练为核心，在数学、代码、逻辑推理上对标 OpenAI o1，且完全开源、支持长思维链与低成本商用，适合科研与工业级复杂推理场景。

核心定位与架构

  • 模型类型：第一代推理专用大模型，基于 Transformer 与混合专家（MoE）架构，总参数量 671B（660B/671B 表述略有差异），激活参数量 37B，上下文窗口 128K tokens，支持中英文双语。
  • 核心版本：
    1. R1-Zero：纯强化学习训练，无监督微调（SFT），展现长思维链与自我验证，但可读性与一致性较弱。
    2. R1：在 R1-Zero 基础上加入冷启动 SFT 与多阶段 RL，解决输出质量问题，性能比肩 o1-preview / 正式版。
  • 开源生态：MIT 许可，开源 660B 的 R1-Zero 与 R1，同时提供基于 Llama/Qwen 架构的 6 个蒸馏小模型（1.5B/7B/14B/32B/70B 等），支持商业应用。

训练与技术创新

  1. 冷启动 + 多阶段 RL 框架：R1 采用四阶段流程 —— 冷启动 SFT（数千高质量思维链样本）→探索 RL（发现有效推理模式）→对齐 RL（提升一致性与可读性）→蒸馏适配小模型，在极少标注下显著提升推理能力。
  2. 长思维链与自我验证：推理过程含反思与验证，思维链长度可达数万字，如 AIME 2025 中新版 R1 平均每题用 23K tokens，准确率由 70% 升至 87.5%。
  3. 纯 RL 驱动推理：R1-Zero 验证 “无需 SFT 仅靠 RL 即可涌现推理能力”，为推理模型训练提供新范式。

性能与优势

维度表现
数学推理AIME 竞赛准确率 70%+（R1-Zero）/87.5%（新版 R1），接近 o1 系列
代码生成复杂逻辑与调试能力突出，支持长代码与多语言编程
通用推理自然语言推理、多步骤问题解决等任务性能对标 o1，且开放完整思考过程
成本优势API 定价：输入 1 元 / 4 元每百万 tokens（缓存命中 / 未命中），输出 16 元每百万 tokens，约为 o1 的 3%

应用与部署

  • 适用场景：数学竞赛、算法研发、代码审计、科学计算、复杂决策辅助、长文本逻辑分析等。
  • 部署方式：
    • 开源使用：GitHub/ModelScope 获取模型权重，支持本地部署（推荐 NVIDIA Ampere 及以上）。
    • 云服务：通过 DeepSeek API 低成本调用，适合快速集成与规模化商用。
  • 版本迭代：2025 年 5 月更新 R1-0528，算力投入增加，思维深度与基准测评表现进一步提升，接近 o3 与 Gemini-2.5-Pro。

小结

R1 以 “纯 RL + 开源 + 长思维链” 打破推理模型的技术与使用壁垒，既适合研究者探索强化学习驱动推理的新方法，也为企业提供低成本、高性能的复杂推理解决方案，推动推理 AI 的开放与普惠。
